# Fluid Mechanics – Notes & Coursework

This repository contains my coursework, lecture materials, and personal study notes from Fluid Mechanics.  
The course covers fundamental fluid behavior, fluid statics, control-volume analysis, internal flow, and computational tools for engineering analysis.

## Repository Structure

### **lectures/**
Lecture slides and reference materials  
– Fundamentals (Ch1)  
– Fluid statics (Ch2)  
– Control volume analysis (Ch5)  
– Internal flow  
– Introduction to CFD  

### **Homework/**
Completed problem sets demonstrating application of fluid mechanics concepts.

### **notes/**
My personal notes and summaries for reviewing key concepts.

## Course Topics
- Fluid properties and classifications  
- Pressure, hydrostatics, and buoyancy  
- Conservation of mass, momentum, and energy  
- Bernoulli equation applications  
- Laminar vs turbulent flow  
- Pipe flow and friction factor  
- Intro to CFD methods and simplified simulation concepts

This repository documents my learning process and foundational understanding of fluid mechanics principles used in engineering analysis.
